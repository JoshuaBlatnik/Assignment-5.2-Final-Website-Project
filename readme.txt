# Tech Savvy Website

## Overview
This is a project for the Tech Savvy website, which provides the latest in technology news, reviews, and tutorials. The project includes the following HTML files:

1. `index.html` - The homepage of Tech Savvy.
2. `about.html` - The About Us page with information about the team, mission, values, and journey of Tech Savvy.
3. `contact.html` - The Contact Us page with a form and additional contact information.

## File Descriptions

### index.html
- This is the main homepage for Tech Savvy.
- Includes sections for Latest News, Subscribe, Featured Reviews, Latest Tech, Top Tutorials, and Community Discussion.
- Features a navigation menu, search bar, and account info button.
- Includes SEO meta tags for better search engine visibility.
- Utilizes external CSS (`style.css`) for styling.
- Utilizes external JavaScript (`script.js`) for dynamic content.

### about.html
- Provides detailed information about Tech Savvy, including the mission, values, team, and journey.
- Structured with multiple sections to enhance readability and user engagement.
- Includes SEO meta tags for better search engine visibility.
- Utilizes external CSS (`style.css`) for styling.
- Utilizes external JavaScript (`script.js`) for dynamic content.

### contact.html
- Contains a contact form for users to get in touch with Tech Savvy.
- Lists additional contact methods such as email, phone, and physical address.
- Includes a Google Map iframe for location visibility.
- Includes SEO meta tags for better search engine visibility.
- Utilizes external CSS (`style.css`) for styling.
- Utilizes external JavaScript (`script.js`) for dynamic content.

## CSS - style.css
- Provides styling for the entire website.
- Includes styles for body, header, navigation menu, sections, and footer.
- Ensures a consistent design across all pages.
- Includes styles for various components like buttons, forms, and social media icons.
- Contains animations for smooth transitions and better user experience.

## JavaScript - script.js
- Adds dynamic functionality to the website.
- Includes an IntersectionObserver to trigger animations when sections come into view.
- Ensures sections and footer become visible smoothly as the user scrolls.

## Social Media Icons
- `facebook-icon.png` - Facebook icon used in the footer and contact sections.
- `twitter-icon.png` - Twitter icon used in the footer and contact sections.
- `instagram-icon.png` - Instagram icon used in the footer and contact sections.

## How to Use
1. Open `index.html` in a web browser to view the homepage.
2. Navigate to `about.html` to learn more about Tech Savvy.
3. Navigate to `contact.html` to get in touch with the Tech Savvy team.
4. Ensure the `style.css` and `script.js` files are in the same directory as the HTML files for proper styling and functionality.
5. Place the social media icons (`facebook-icon.png`, `twitter-icon.png`, `instagram-icon.png`) in the same directory as the HTML files for the icons to display correctly.

## Additional Information
- This project is designed to be responsive and should work well on various devices and screen sizes.
- SEO meta tags are included to improve search engine visibility and social media sharing.
- External CSS and JavaScript files are used to keep the code organized and maintainable.
